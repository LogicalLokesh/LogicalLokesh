# Hey, I'm Lokesh

CS student. I build things — software, hardware, and the infrastructure in between.
Equal parts code and silicon. I care about understanding systems, not just using them.

---

## 🔧 What I'm Into

**Self-hosted everything** — Running my own media server, NAS, DNS, and a dozen Docker services on a home server I built from scratch. If there's a cloud alternative, I've probably replaced it.

**Hardware** — Built and tuned my own workstation. I pick parts, assemble, overclock, and obsess over airflow and thermals.

**Audio** — Originals-only library. No streaming subscriptions. I own my music, and I sometimes remaster tracks the way I think they should sound.
